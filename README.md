

# WeatherApp 

IOS application written using Swift 4.2. It uses CLEAN architecture.


### Prerequisites
Xcode 10.1
Swift 4.2
Deployment Target 11.0


## Feature
* 
* Current day and next five days weather report (api call)
* Offline storage
* Pull to refresh
* Api call after every 5 mins
* Error handeling with toast
